This function Locks a specific pin (Can't change its Mode anymore)

Parameters
copy_u8PORTRefereces to the port .. options(GPIOA, GPIOB, GPIOC)
copy_u8PINPin Number ... options(PIN0, PIN1, ....., PIN15)

This Function Locks a specific Pin so that no one Can Change the mode of that pin anymore

Then Cofirm The lock using GPIOx_LCKR register

LOCK key writing sequence(According to Data Sheet):

Write 1

Write 0

Write 1

Read 0

Read 1 (this read is optional but confirms that the lock is active)

End of Lock sequence

uint32  copy_u8PortMask

This function Locks a Port by Mask.

Parameters
copy_u8PORTRefereces to the port .. options(GPIOA, GPIOB, GPIOC)
<pre>copy_u8PortMaskPort Mask that the users chooses which pins to be locked and which to be not it consists of 16 bits each bit represents a pin from the 16 pins of the port if bit 0 then the coresponding pin will not be lokced if bit 1 then the coresponding pin will be locked

This Function Locks a port by mask so that no one Can Change the mode of the specified pins anymore

First.. Lock the Pins using mask

Then Cofirm The lock using GPIOx_LCKR register

LOCK key writing sequence(According to Data Sheet):

Write 1

Write 0

Write 1

Read 0

Read 1 (this read is optional but confirms that the lock is active)

End of Lock sequence

Parameters

copy_u8PINPin Number ... options(PIN0, PIN1, ....., PIN15)
copy_u8Valuevalue that should be applied to the required pin
Precondition
You should Call GPIO_voidSetPinDirection() in first
 Please Note that
     In GPIOx_BSRR regsiter(From bit 0 to 15) puting
         0: No action on the corresponding ODRx bit(Pin)
         1: Set the corresponding ODRx bit(Pin)

     In GPIOx_BRR regsiter(From bit 0 to 15) puting
         0: No action on the corresponding ODRx bit(Pin)
         1: Reset the corresponding ODRx bit(Pin)
